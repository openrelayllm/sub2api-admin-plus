# Release Notes

## v0.37.0 - 2026-07-04

### 新增

- 新增运营看板 `/admin/kanban`，聚合市场价格、缓存效率、供应质量、接入验收、模型利润和看板事件。
- 新增市场价格、缓存效率、供应质量、接入验收和看板事件数据表，快照采用追加写入，保留历史证据。
- 新增公开价格文本解析、公开价格 URL 导入和价格源发现能力，支持识别套餐、充值赠送、最低充值门槛、倍率、限时活动和模型上下架信号。
- 新增接入验收证据调度与回填，支持通过 scheduler run 生成供应质量、缓存效率和最终验收报告。
- 新增看板事件到动作建议的生成链路，开放事件可转为暂停、降权、切换、凭据复核或定价复核建议。
- 新增已审批动作建议执行接口和执行记录，支持对供应商暂停/降权/切换类操作写入执行回执。

### 改进

- 动作推荐页展示执行记录和执行入口，运营可以在审批后直接落地可自动执行的建议。
- 调度服务补充看板接入验收任务识别、纯度检测证据写入和完成后回填流程。
- NewAPI 会话适配增强 access token 与用户信息识别，便于供应商后台直登和自动化操作。
- 站点发现服务补充看板价格源发现依赖，运营可从站点目录快速定位公开价格页候选。
- 补充市场价格与供应质量看板 PRD 到 `docs/roadmap/kanban/README.md`。

### 修复

- 修正动作建议只能生成但缺少执行闭环的问题。
- 修正接入验收证据任务完成后无法回填为结构化报告的问题。

### 测试

- 增加看板服务、市场价格解析、接入验收证据回填、动作建议执行和调度回归测试。
- 补充 Admin Plus 路由表面测试和前端运营路由测试，覆盖新增看板入口。

### 发布

- 更新版本号到 `0.37.0`。
- GitHub Release 继续只发布 Linux 产物：`linux_amd64`、`linux_arm64` 和 `checksums.txt`。
- DockerHub/Railway 镜像渠道不随常规发布自动执行；如需要，单独走镜像渠道发布流程。
