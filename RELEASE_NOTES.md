# Release Notes

## v0.36.0 - 2026-07-03

### 新增

- 新增账号倍率同步后端服务，支持按本地 API Key 账号反查供应商 Key、匹配当前倍率、记录同步历史并按匹配结果更新本地账号名称。
- 新增账号倍率同步管理页，支持 OpenAI/Anthropic 切换、批量同步、单账号重试、批量更新名称、单账号更新名称和清空同步历史。
- 新增账号倍率同步历史表 `admin_plus_account_rate_sync_history`，记录本地账号、匹配供应商、倍率、目标名称、状态、错误信息和改名结果。
- 新增独立倍率检测页面，将供应商通道倍率/通畅检测从供应商列表拆出，支持协议切换、最佳/全部范围、复测、绑定本地分组和调度控制。

### 改进

- 运营侧导航新增“倍率检测”和“账号倍率同步”入口，历史倍率监控路径重定向到新的倍率检测页。
- 供应商列表移除内嵌倍率检测面板，降低供应商管理页复杂度。

### 修复

- 修正倍率检测入口长期复用供应商列表导致的页面职责混杂问题。

### 测试

- 补充 Admin Plus 路由表面测试，覆盖账号倍率同步 API 和新增前端运营路由。

### 发布

- 更新版本号到 `0.36.0`。
- GitHub Release 继续只发布 Linux 产物：`linux_amd64`、`linux_arm64` 和 `checksums.txt`。
- DockerHub/Railway 镜像渠道不随常规发布自动执行；如需要，单独走镜像渠道发布流程。
