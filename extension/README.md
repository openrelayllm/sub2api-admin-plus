# Sub2API Admin Plus Chrome Extension

这是 Admin Plus 的最小 Chrome MV3 执行器，用于领取后端插件任务、按任务租约读取供应商网页登录凭据、打开供应商后台并执行页面采集。

当前实现不生成 mock 数据：

- 能从页面解析到费率、余额、优惠、账单或并发数据时，才调用任务完成接口。
- 页面未登录时会尝试使用临时 token 或账号密码登录。
- 页面结构不支持或没有解析到必要字段时，会调用任务失败接口。

使用方式：

1. 打开 Chrome `chrome://extensions`。
2. 启用开发者模式。
3. 加载 `extension/` 目录。
4. 在已登录的 Admin Plus 后台页面点击“读取当前后台 Token”。
5. 点击“领取并执行”。

后续每个供应商需要按真实后台页面补专用适配器，避免通用 DOM 解析误判。
